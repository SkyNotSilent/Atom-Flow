# AtomFlow（原子流）- 对外发布文档

> 让每一篇看过的知识，都成为可复用的创作资产。

## 我们在解决什么问题

很多人每天读了很多内容，但最终只剩“看过了”。  
AtomFlow 要解决的是：**如何把阅读转成可复用的知识积累**，并在写作时快速调用。

## 一句话介绍

AtomFlow 是一款面向内容创作者、产品经理和知识工作者的知识工作台，提供「今日推送 → 存入知识库 → 原子卡片 → 魔法写作」的完整链路。

## 核心场景

- **高质量输入**：聚合信息源，快速浏览并进入全文
- **知识沉淀**：一键存入，拆解为原子卡片
- **内容复用**：按观点/数据/金句/故事组织素材
- **写作输出**：在写作场景中召回和组合卡片

## 产品能力

### 今日推送
- 多来源内容流聚合
- 按来源查看与阅读
- 存入知识库时提供阶段化反馈

### 我的知识库
- 原子卡片统一类型：
  - 观点
  - 数据
  - 金句
  - 故事
- 可按类型和来源筛选
- 可回溯引用到原文

### 魔法写作
- 召回并组织卡片素材
- 快速生成草稿
- 支持复制与导出

## 与传统收藏夹的区别

- 收藏夹是“存链接”，AtomFlow 是“存结构化知识”
- 收藏夹解决“以后再看”，AtomFlow 解决“现在可用”
- 收藏夹难复用，AtomFlow 为复用而设计

## 当前版本亮点

- 三栏联动工作区，支持栏宽拖拽
- 知识类型统一（观点/数据/金句/故事）
- 来源模式与原子模式双视图
- 卡片支持引用回溯原文

## 适用人群

- 做内容但不想反复从零起稿的人
- 需要持续输入并沉淀方法论的人
- 想建立个人知识资产体系的人
